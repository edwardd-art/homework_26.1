# 🛒 Управление продуктами - Django CRUD приложение

Веб-приложение для управления каталогом продуктов с полным CRUD функционалом и валидацией данных.

## 📋 Описание проекта

Проект представляет собой веб-приложение на Django для управления продуктами. Реализованы все операции CRUD (создание, чтение, обновление, удаление) с дополнительной валидацией данных.

### 🎯 Функциональность

- ✅ Создание продуктов
- ✅ Просмотр списка продуктов
- ✅ Редактирование продуктов
- ✅ Удаление продуктов
- ✅ Валидация названия и описания (запрещенные слова)
- ✅ Валидация цены (не может быть отрицательной)
- ✅ Стилизованные формы с Bootstrap
- ✅ Административная панель Django

### 🚫 Запрещенные слова

При создании или редактировании продукта проверяется отсутствие следующих слов (в любом регистре):
- казино
- криптовалюта
- крипта
- биржа
- дешево
- бесплатно
- обман
- полиция
- радар

### 📊 Модель Product

| Поле | Тип | Описание |
|------|-----|----------|
| name | CharField | Название продукта |
| description | TextField | Описание продукта |
| price | DecimalField | Цена (не может быть отрицательной) |
| is_available | BooleanField | В наличии/нет |
| created_at | DateTimeField | Дата создания |
| updated_at | DateTimeField | Дата обновления |
