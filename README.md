# 🛒 Интернет-магазин - Управление продуктами и пользователями

Веб-приложение на Django для управления продуктами с полной аутентификацией пользователей.

## 📋 Описание проекта

Проект представляет собой интернет-магазин с системой управления продуктами и пользователями. Реализованы CRUD операции для продуктов, аутентификация пользователей с авторизацией по email, отправка приветственных писем.

### 🎯 Функциональность

#### Для всех пользователей:
- ✅ Просмотр списка продуктов
- ✅ Регистрация с подтверждением email
- ✅ Авторизация по email и паролю

#### Для авторизованных пользователей:
- ✅ Создание продуктов
- ✅ Редактирование продуктов
- ✅ Удаление продуктов
- ✅ Управление профилем

### 🚫 Валидация продуктов
- Запрещенные слова в названии и описании
- Цена не может быть отрицательной

### 👤 Модель пользователя
- Авторизация по email (вместо username)
- Дополнительные поля: аватар, телефон, страна

## 🛠️ Технологии

- **Python** 3.10+
- **Django** 4.2.7
- **Bootstrap** 5.3.2
- **SQLite** (разработка)
- **Pillow** (для работы с изображениями)
