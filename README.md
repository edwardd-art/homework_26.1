# 🛒 Интернет-магазин - Управление продуктами с кешированием

Веб-приложение на Django с полным CRUD, аутентификацией, правами доступа и кешированием через Redis.

## 📋 Описание проекта

Интернет-магазин с системой управления продуктами, пользователями, ролями и кешированием для оптимизации производительности.

### 🎯 Функциональность

#### Для всех пользователей:
- ✅ Просмотр списка продуктов (с кешированием)
- ✅ Просмотр продуктов по категориям (с кешированием)
- ✅ Детальная страница продукта (с кешированием)
- ✅ Регистрация с подтверждением email
- ✅ Авторизация по email и паролю

#### Для авторизованных пользователей:
- ✅ Создание продуктов (автоматическое назначение владельца)
- ✅ Редактирование своих продуктов
- ✅ Удаление своих продуктов
- ✅ Публикация своих продуктов

#### Для модераторов:
- ✅ Редактирование любых продуктов
- ✅ Удаление любых продуктов
- ✅ Отмена публикации любых продуктов

### 🚀 Кеширование с Redis
- Кеширование списка всех продуктов (`products_all`)
- Кеширование продуктов по категориям (`category_{category}`)
- Кеширование деталей продукта (`product_{id}`)
- Кеширование списка категорий (`categories_list`)
- TTL: 15 минут (настраивается)
- Автоматическая очистка кеша при изменениях

### 🚫 Валидация продуктов
- Запрещенные слова в названии и описании
- Цена не может быть отрицательной

### 👤 Модель пользователя
- Авторизация по email (вместо username)
- Дополнительные поля: аватар, телефон, страна

## 🛠️ Технологии

- **Python** 3.10+
- **Django** 4.2.7
- **Bootstrap** 5.3.2
- **SQLite** (разработка)
- **Redis** (кеширование)
- **Pillow** (работа с изображениями)
